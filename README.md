# simpletodocli
Simple To-Do CLI — это простой менеджер задач для терминала, написанный на Python. Он позволяет добавлять задачи, просматривать список, отмечать задачи выполненными и удалять их.  Проект создан как небольшой CLI-инструмент для практики Python и работы с аргументами командной строки.
